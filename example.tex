\section{Numbered examples in Markdown}

Numbered examples can look like this:

\ex. Example one\label{ex:example:1}

\ex. Example two\label{ex:example:2}

We can also embed alphanumeric lists within example lists

\ex. Let's embed. \a. Embedded example one \b. Embedded example two.

The period syntax works as well:

\ex. Example four.\label{ex:example:four}

We can reference \ref{ex:example:four} and \ref{ex:example:2}, and also
forward \ref{ex:example:last}. Even with some intervening text.

\ex. Is two.\label{ex:example:last}

Regular numbered lists keep their own numbering.

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  First list item
\item
  Second list item
\end{enumerate}

But maybe this is confusing. Anyway, we can refer back to
\ref{ex:example:1}.
